# OVERVIEW
A secure order management system built to demonstrate
full-stack JavaScript development with enterprise practices.

# TECH STACK    
- React 
- NodeJS (Expres)
- MySQL
- JWT Authenication
- Docker Compose

# Features
- Secure login with JWT 
- Role-based authorization
- Transaction-safe order processing
- Online Payment

# Getting Started
1. Clone repository
2. Run `docker-compose up`
3. Access API at http://localhost:3003
